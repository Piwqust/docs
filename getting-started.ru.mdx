— как это работает

1. выберите слои, которые нужно переименовать
2. подключите ИИ-провайдера (вставьте бесплатный api ключ или используйте lm studio)
3. нажмите зелёный круг — ИИ создаст осмысленные имена с учётом ваших настроек

— установка

<Steps>
  <Step title="установите из сообщества figma">
    найдите **layersense** в сообществе figma и нажмите установить.
  </Step>
  <Step title="запустите плагин">
    нажмите `cmd + /` (или `ctrl + /` на windows) и введите "layersense" для запуска.
  </Step>
  <Step title="получите бесплатный api ключ">
    по умолчанию используется cerebras. получите бесплатный ключ на [cloud.cerebras.ai](https://cloud.cerebras.ai) — пошагово в [руководстве по провайдерам](/providers).
  </Step>
  <Step title="вставьте ключ и переименуйте">
    нажмите шестерёнку настроек, перейдите на вкладку **api**, вставьте ключ, затем выберите слои и запускайте переименование.
  </Step>
</Steps>

<Note>
в плагине уже встроен стандартный ключ с ограниченными лимитами на старте. если он не отвечает или исчерпаны лимиты, вставьте свой ключ: откройте настройки → вкладка **api** → выберите провайдера → вставьте ключ → сохраните. инструкции по получению ключа для каждого провайдера смотрите в разделе [провайдеры](/providers).
</Note>

— быстрые советы

<AccordionGroup>
  <Accordion title="сначала выберите слои" icon="mouse-pointer">
    выберите хотя бы один слой перед нажатием на переименование. плагин работает со стандартными именами figma вроде "frame 1", "rectangle 12", "text 5".
  </Accordion>
  <Accordion title="начните с малого" icon="seedling">
    начните с одного фрейма или небольшой группы, чтобы увидеть, как ИИ интерпретирует структуру. затем масштабируйтесь.
  </Accordion>
  <Accordion title="проверьте иерархию" icon="sitemap">
    ИИ точнее работает, когда видит иерархию (например, кнопки внутри контейнеров). чистая структура = лучшее качество имён.
  </Accordion>
  <Accordion title="проверьте api соединение" icon="plug">
    если после нажатия ничего не происходит, проверьте ключ и endpoint в настройках или пройдите [руководство по провайдерам](/providers).
  </Accordion>
</AccordionGroup>

— устранение неполадок

<Warning>
**нет слоёв для переименования?** значит, выбранные слои имеют пользовательские имена или скрыты/заблокированы. выберите фрейм с дочерними элементами со стандартными именами.
</Warning>

| проблема | решение |
|----------|----------|
| "выберите слои для переименования" | выберите хотя бы один слой перед нажатием кнопки переименования |
| ничего не происходит | проверьте ваш api ключ в настройках → вкладка api |
| ошибки лимита скорости | плагин автоматически повторяет с задержками; подождите немного |
| медленный ответ | попробуйте более быструю модель: `gpt-oss-120b`, `gemini-2.5-flash`, `qwen-3-32b` |
| имена не соответствуют ожиданиям | настройте фокус, детализацию или добавьте пользовательские правила |
