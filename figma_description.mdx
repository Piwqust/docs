## full description

**layersense** uses ai to automatically rename your figma layers with meaningful, semantic names.

**— what it does**

select your messy layers ("frame 45", "rectangle 12", "group 7") and click one button. the ai analyzes the content, hierarchy, and context to generate proper names like "hero section", "submit button", or "user avatar".

**— 100% free**

no subscription. no trial. no "free for 10 layers". layersense works with free ai providers:

— (1) cerebras — free api with blazing-fast inference (3,000+ tok/s)
— (2) google gemini — generous free tier at aistudio.google.com
— (3) openrouter — free models available
— (4) lm studio — run locally on your own computer (complete privacy)

bring your own api keys. we don't have a backend — your data goes directly from figma to the ai provider you choose.

**— supported models (december 2025)**

— cerebras: gpt-oss-120b, llama-3.3-70b, qwen-3-32b (3,000+ tok/s, free)
— openai: gpt-5.1, gpt-5-nano, gpt-4.1-mini
— anthropic: claude-sonnet-4-5, claude-haiku-4-5, claude-opus-4-5
— google: gemini-3-pro, gemini-2.5-flash, gemini-2.5-pro (1M context)
— local: any model via lm studio

**— customization**

— naming styles: title case, camelcase, kebab-case, snake_case
— output languages: english, russian, or auto-detect
— focus: function, content, or context-based naming
— word count: set min/max words per name
— custom rules: "always prefix buttons with btn-"

**— privacy**

direct api calls. no middleman. your designs never touch our servers (we don't have any). with local models, your data never leaves your computer.

**— how to start**

1. install layersense
2. get a free api key at cloud.cerebras.ai
3. paste key in settings → api tab
4. select layers and click the green circle

documentation: layersense.mintlify.app
