— api providers & free access

layersense is "bring your own key", meaning you are not locked into a subscription.

— how to run 100% free

you don't need a credit card. here is where to get free access:

1. **google gemini** (recommended) • **get key**: [aistudio.google.com](https://aistudio.google.com/) • **model**: `gemini-1.5-flash` (fastest) or `gemini-2.0-flash-exp`. • **note**: high rate limits, perfect for heavy usage.
2. **cerebras** • **get key**: [cloud.cerebras.ai](https://cloud.cerebras.ai/) • **model**: `llama3.1-70b` • **note**: incredibly fast, though context may be limited during peak traffic.
3. **openrouter** • **get key**: [openrouter.ai](https://openrouter.ai/) • **free models**: `mistralai/mistral-7b-instruct:free`, `meta-llama/llama-3-8b-instruct:free`.
4. **local llms** (lm studio) • **download**: [lmstudio.ai](https://lmstudio.ai/) • **setup**:
   1. install lm studio.
   2. load a model (e.g., `qwen 2.5` or `llama 3.2`).
   3. start the **local inference server**.
   4. in layersense, select **local (lm studio)**. • **note**: zero data leaves your computer.

— commercial providers

if you have a key, we also support: • **openai**: `gpt-4o`, `gpt-4o-mini` • **anthropic**: `claude-3-5-sonnet`, `claude-3-haiku`

— settings configuration

• **model selection**: manually type any model id (e.g., `gpt-4o`). • **temperature**: controls "creativity". lower (0.1) is more consistent; higher (0.7+) is more random. • **export/import**: save your settings to a json file to share with teammates.
